## Verilog implementation of circuits listed below :
- `n-bit carry-look-ahead full-adder`
- Simple `shift-register`

## Usage :
```bash
## compile command :
iverilog -o [DESTINATION_FILE] [VERILOG_FILE.v]

## execution command :
vvp [DESTINATION_FILE]
```
